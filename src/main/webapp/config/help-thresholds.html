Max Messages

  • For max messages the Threshold indicates that the build must have less or equal messages than the value specified.
  • Improvement indicates that the amount of messages must be less or equal to the number of messages found in previous build.
  • The default value None basically means that this field is ignored.

Project Compliance Index

  • Threshold is the lowest compliance level allowed for this project, anything lower than the percentage value will mark the build as unstable
  • Improvement means that the new build must have better compliance than before, ie. a higher project compliance percentage.
  • The default value None basically means that this field is ignored.

File Compliance Index

  • Threshold is the lowest compliance level allowed for this project, anything lower than the percentage value will mark the build as unstable
  • Improvement means that the new build must have better compliance than before, ie. a higher file compliance percentage.
  • The default value None basically means that this field is ignored.
